Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и порядок их вызовов.

```go
package main

import (
    "fmt"
)

func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}


func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}


func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}
```

Ответ:
```
2
1

defer добавляет в стек выполнения отложенную функцию, которая выполнится перед выходом из основной функции. Порядок вызовов происходит в обратном порядке их добавления (LIFO). Оператор return устанавливет возвращаемое значение перед выполенением отложенной функции. Так как в функции test возвращаемое значение именованное, то отложенная функцию сможет инкрементировать возвращаемое значение. В функции anotherTest возвращаемое значение создается в самой функции и return установит это значение перед выполнением отложенной функции.
```
